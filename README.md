# CraveDelivery

> CraveDelivery is a robust, React-based food delivery platform connecting gourmets with top-tier restaurants. Experience seamless ordering, real-time menus, and trusted reviews in a responsive, modern interface designed for speed and delight.

## About
CraveDelivery is a premium food delivery application built to provide a seamless connection between hungry customers and the best local restaurants. With a focus on user experience, the platform offers an intuitive interface for browsing menus, placing orders, and tracking deliveries.

## Features
-   **Gourmet Selection**: Curated list of top-rated restaurants.
-   **Seamless Ordering**: Easy-to-use cart and checkout process.
-   **Real-Time Menus**: Up-to-date pricing and availability.
-   **Trusted Reviews**: Verified customer feedback to help you choose.
-   **Responsive Design**: optimized for both desktop and mobile devices.

## Tech Stack
-   **Frontend**: [React](https://react.dev/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Routing**: [React Router](https://reactrouter.com/)
-   **Styling**: Custom CSS with responsive utilities.

## Getting Started

To run CraveDelivery locally, ensure you have [Node.js](https://nodejs.org/) installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/cravedelivery.git
    cd cravedelivery
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the application:**
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser to view the app.

## Project Structure
-   `src/components`: Reusable UI components (Header, Hero, Features, etc.).
-   `src/pages`: Main application views (Home, Menu, Order, etc.).
-   `src/context`: React context for global state management.
-   `src/assets`: Images and static resources.
